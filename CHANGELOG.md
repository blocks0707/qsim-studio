# Changelog

## [0.1.0] - 2025-03-08

### Added
- OpenQASM 2.0 syntax highlighting and language support
- Qiskit Python code intelligence (completion, hover, diagnostics)
- Circuit visualization (interactive SVG viewer)
- Simulation execution via qsim-cluster backend
- Result viewer with histogram and state vector display
- Algorithm registry with 12 built-in quantum algorithms
- AI assistant integration via OpenClaw
- Job management tree view
- Quantum circuit parser for QASM and Qiskit
- Unit test suite with Jest
